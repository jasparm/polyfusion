Requirements:
 - npm
 - vite
 - three.js

How to run:
 - Run command "npx vite" in terminal. 
 - Go to address given.

Controls:
 - First three numbers spawn a shape:
    * 1 spawns a cube
    * 2 spawns a sphere
    * 3 spawns a cone

 - The shapes can also be moved by pressing Q
    * T sets object to be transformed
    * R sets the object to be rotated
    * X enables axis helpers.
The shapes can then be moved using the mouse or arrow keys.

Space enables and disables a spinning animation.
